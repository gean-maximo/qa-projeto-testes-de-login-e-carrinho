# Casos de Teste - Login 

## CT001 - Login válido

### Passos 
1. Abrir o site SauceDemo
2. Informar usuário valido
3. Informar senha válida
4. Clicar em Login

### Resultado esperado
Usuario deve acessar a pagina de produtos.

---

## CT002 - Senha inválida

### Passos
1. Abrir o site SauceDemo
2. Informar usuário válido
3. informar senha inválida
4. Clicar em Login

### Resultado esperado
Sitema deve exibir mensagem de erro

---

## CT003 - Campos vazios

### Passos
1. Abrir o site Sauce demo
2. Não preencher usuário
3. Não preencher senha
4. Clicar em login

### Resultado esperado
Sistema deve impedir o login.
