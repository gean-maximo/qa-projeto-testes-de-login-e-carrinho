# Casos de Teste - Carrinho

## CT004 - Adicionar produto ao carrinho

### Passos 
1. Fazer login no sistema
2. Selecionar um produto
3. Clicar em "Add to cart"

### Resultado esperado
O produto deve ser adicionado ao carrinho

---

## CT005 - Remover produto do carrinho

### Passos
1. Fazer login no sistema
2. Adicionar um produto ao carrinho
3. Clicar em "Remove"

### Resultado esperado
O produto deve ser removido do carrinho.

---

##CT006 - Adicionar multiplos produtos

### Passos 
1. Fazer login no sistema
2. Adicionar dois produtos ao carrinho
3. Abrir o carrinho

### Resultado esperado
Os dois produtos devem aparecer no carrinho.
